# Clipboard Manager
## Description
A lightweight desktop application that automatically records everything you copy and stores it in a searchable, organised history. Instead of relying on a single clipboard slot, the app keeps a running list of your recent snippets so you can quickly find, pin, and reuse text at any time. It runs quietly in the background, captures new clipboard content in real time, and provides a fast interface for browsing and copying items back to the system clipboard.
